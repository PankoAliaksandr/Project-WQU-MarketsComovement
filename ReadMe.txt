1. Python 2.7 (Anaconda 2 x64) 'Spyder' environment
2. OS: Windows 7 x64
3. Libraries Used:
	a) datetime
	b) pandas
	c) pandas_datareader (requires installation):
	   Anaconda Prompt: pip install pandas-datareader
	d) matplotlib (requires installation):
	   Anaconda Prompt:pip install matplotlib
	e) dateutil


4. To run program:
	a) Make sure the file 'my data.csv' in the same folder as the main one
	b) Open a prompt (a Windows 'cmd' ): start > run > "cmd" 
	c) Navigate to where the python file is, using the commands 'cd' (change directory)
	   and 'dir' (to show files in the directory, to verify your head). For our example something like,
	d) Run the program: python Panko_Aliaksandr-Project_2-Nov_13_2017.py

5. Alternative way: just open the file from a Spyder and press F5
